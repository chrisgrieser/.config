## Config structure

```text
├── lua
│  ├── config # keybindings, options, …
│  ├── funcs # utility functions
│  └── plugins # plugins & their configs
├── after
│  └── ftplugin # filetype-specific configs
├── mac-helper # for neovide users on macOS
├── snippets
│  ├── basic
│  └── project-specific
└── templates
```
