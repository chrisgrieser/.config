# Drink reminder

## Installation
1. Unzip the file.
2. In Obsidian, go to `Settings` → `Community Plugins`.
3. Press the folder button, the folder `{obsdian-vault-path}/.obsidian/plugins`
   will open. Move unzipped folder `drink-reminder` into this folder.
4. Restart Obsidian.
5. *(only when installing)* enable the plugin `Drink Reminder`.

## Update
1. In Obsidian, go to Settings → Community Plugins.
2. Press the folder button, the folder `{obsidian-vault-path}/.obsidian/plugins`
   will open. Replace the `main.js` file.
3. Restart Obsidian.

## Usage
Will automatically remind you every hour to drink. Nothing to configure or to do
otherwise.
